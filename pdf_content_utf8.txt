CASE STUDY: AI-Based Smart Complaint Management System 
Design and implement a MERN Stack application that allows users to register complaints online. The 
system should use AI APIs to classify complaint priority, generate automated responses, and 
recommend the concerned department. 
The application should support: 
• Complaint registration  
• Complaint tracking  
• AI-based complaint categorization  
• Secure authentication  
• Deployment on Render.
